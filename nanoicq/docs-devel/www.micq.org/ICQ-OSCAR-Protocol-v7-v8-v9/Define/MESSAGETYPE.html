#define MESSAGETYPE

The messages types used. For type that consists of several parts, these parts are seperated by a 254 = 0xfe = þ character.

MSG_AUTO 0x0000 An automatic message.
MSG_NORM 0x0001 A plain normal message.
MSG_CHAT 0x0002 A chat request.
MSG_FILE 0x0003 A file transfer request.
MSG_URL 0x0004 An URL message. The message consists of the description and the url.
MSG_AUTHREQ 0x0006 An authorization request.
MSG_AUTHREJ 0x0007 An authorization reject message.
MSG_AUTHACC 0x0008 An authorization accept message.
MSG_ADDED 0x000c = 12 You were added to the sender's contact list.
MGS_WEB 0x000d = 13 A message sent through www.icq.com's web pager.
MSG_EMAIL 0x000e = 14 A message sent through the email pager ([uin]@pager.icq.com).
MSG_CONTACT 0x0013 = 19 A contact list message.
MSG_EXTENDED 0x001a = 26 An extended message. The packet will contain more data.
MSG_GET_AWAY 0x03e8 = 1000 A message requesting the "away" auto message.
MSG_GET_OCC 0x03e9 = 1001 A message requesting the "occupied" auto message.
MSG_GET_NA 0x03ea = 1002 A message requesting the "not available" auto message.
MSG_GET_DND 0x03eb = 1003 A message requesting the "do not disturb" auto message.
MSG_GET_FFC 0x03ec = 1004 A message requesting the "free for chat" auto message.
MSGF_MASS 0x8000 The message was sent to several recipients.
