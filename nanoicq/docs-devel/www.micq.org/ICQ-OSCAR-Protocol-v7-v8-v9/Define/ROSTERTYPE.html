#define ROSTERTYPE

The different types of entries in a (server-side) roster.

ROSTER_CONTACT 0x0000 a normal contact
ROSTER_GROUP 0x0001 a group of contacts
ROSTER_VISIBLE 0x0002 a contact on the visible list
ROSTER_INVISIBLE 0x0003 a contact on the invisible list
ROSTER_VISIBILITY 0x0004 this entry contains visibility setting TLV(0xca)=TLV(202), is usually named "" and is found in the meta group #0.
ROSTER_9 0x0009 unknown; this entry contains TLV(0xcd)=TLV(205) which contains a string like "2586,0,0,0,60". This entry is usually named "ICQTIC", found in the meta group #0 and created by ICQ2go.
ROSTER_IGNORE 0x000e = 14 a contact on the ignore list
ROSTER_11 0x0011 = 17 unknown; it is usually named " ", found in the meta group #0 and empty.
ROSTER_IMPORT 0x0013 = 19 this entry contains the import time TLV(0xd4)=TLV(212), is usually named "IMPORTTIME" and is found in meta group #0.
