PEER_INITACK

Channel: 0

Acknowledges the receipt of a PEER_INIT packet.

Parameters

Data type Content Name Description
BYTE 01 CMD The command: acknowlegde the PEER_INIT.
3 00 00 00 UNKNOWN Unknown: empty.

Examples

01:48:41 Eingehendes TCP-Paket: (53)*
         01 00 00 00                                             ....
