PEER_FILE_STARTACK

Channel: 0

This packet is sent to acknowledge a starting file transfer and is sent as a response to PEER_FILE_START. Note: This is the same as v6 of the file transfer protocol.

Parameters

Data type Content Name Description
BYTE 03 CMD The command: PEER_FILE_STARTACK.
DWORD.L xx xx xx xx OFFSET The offset within the file from where to start transfer. Used to be able to resume transfer.
DWORD 00 00 00 00 EMPTY Unknown: empty.
DWORD.L xx xx xx xx SPEED The receiver's speed again.
DWORD.L xx xx xx xx NUMBER The number of this file in the batch.

Examples

     0000: 03 00 00 00 00 00 00 00  00 64 00 00 00 01 00 00   .........d......
     0010: 00                                                 .
