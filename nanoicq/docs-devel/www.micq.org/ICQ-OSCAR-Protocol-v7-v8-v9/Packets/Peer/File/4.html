PEER_FILE_STOP

Channel: 0

Request to stop the current file transfer and continue with the next file to send. Note: this is the same as v6 of the file transfer protocol.

Parameters

Data type Content Name Description
BYTE 04 CMD The command: PEER_FILE_STOP.
DWORD.L xx xx xx xx NUMBER Posibly the file number to abort.

Examples

No examples.
