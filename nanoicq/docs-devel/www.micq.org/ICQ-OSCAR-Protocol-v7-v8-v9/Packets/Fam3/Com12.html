SRV_USEROFFLINE

Channel: 2 SNAC(3,12)

This is sent when a user goes offline.

Parameters

Data type Content Name Description
BUIN ... UIN The UIN of the contact that goes offline.
DWORD 00 00 00 01 NULL Unknown: empty.
TLV(1) 00 01 00 02 00 00 UNKNOWN Unknown: 0.

Examples

2A 02 B4 BA 00 1E                      FLAP
   00 03 00 0C 00 00 81 83 9B 56       SNAC
      09                               Length = 9
         31 31 32 33 34 35 36 37 38    UIN = 112345678
      00 00 00 01                      Unknowns
      00 01 00 02                      TLV(1)
         00 00                         0
