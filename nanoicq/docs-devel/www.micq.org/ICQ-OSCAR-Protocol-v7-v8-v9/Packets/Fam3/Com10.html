SRV_REFUSED

Channel: 2 SNAC(3,10)

A contact was refused from being added to the contact list. Ooops!

Parameters

Data type Content Name Description
BUIN xx ... UIN The UIN refused from being added to the contact list.

Examples

13:32:05 Eingehendes Server-Paket (Version 8):
  2a 02 3c 7d 00 13               FLAP  ch 2 seq 00003c7d length 0013
  00 03 00 0a 00 00 92 cb  03 ac  SNAC     (3,a) [unknown] flags 0 ref 92cb03ac
  08 31 37 35 30 32 31 35  31                       '.17502151'
