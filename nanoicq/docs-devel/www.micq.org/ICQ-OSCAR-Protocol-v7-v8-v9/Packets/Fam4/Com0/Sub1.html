SUB_MSG_TYPE1

Channel: 2 SNAC(4,0) 1

The content of the TLV of messages in format 1.

Parameters

Data type Content Name Description
TLV(1281) 05 01 xx xx xx CAPABILITIES Each byte describes a capability-like feature of the message. Usually 01 is sent, or 01 and 06, if the receiver has CAP_UTF-8. In the latter case, the message may be encoded in UCS-2BE (or UTF-16?), otherwise the local encoding will be used and specified.
TLV(257) 01 01 xx xx ... MESSAGE This TLV encapsulates the message.
WORD.B xx xx ENCODING The encoding used for the message. 
0x0000: US-ASCII
0x0002: UCS-2BE (or UTF-16?)
0x0003: local 8bit encoding, eg iso-8859-1, cp-1257, cp-1251.
Beware that UCS-2BE will contain zero-bytes for characters in the US-ASCII range.
WORD.B xx xx UNKNOWN Unknown; seen: 0x0000 = 0, 0xffff = -1.
STR ... MESSAGE The message in the afore specified encoding.
