NanoICQ

Description

NanoICQ is a small and highly portable ICQ library and client written in pure Python. Current version of GUI uses wxPython, based on wxWidgets C++ library.

Status

It's not yet released as standalone package, however you can download sources using CVS or SVN repositary to look at the current state.
